//  SMART HOME MANAGEMENT SYSTEM  //

A full-stack web application that enables homeowners to monitor, control, and manage smart devices efficiently. The system supports multiple user roles such as Admin, Homeowner, and Technician.

Frontend
-React.js
-Tailwind CSS
-Axios

Backend
-Java
-Spring Boot 
-Spring Security(JWT Authentication)
-JPA / Hibernate
-Maven

Database
-MySQL

//  User Roles  //

-Admin
    1.Manage users
    2. Monitor system activity
    3. Approve technicians

-Homeowner
    1. Add & manage smart devices
    2. Monitor energy usage
    3. Control devices remotely

-Technician
    1. Maintain devices
    2. View assigned service requests
